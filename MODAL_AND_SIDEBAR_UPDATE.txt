üéØ PROJECT OVERVIEW
================================================================================
A comprehensive AWS-style documentation website for MyIDEX HUB Standard Operating
Procedures (SOPs). Features 41 fully documented sections across 5 modules with
advanced UI components, image viewer, search, animations, and responsive design.

Main Stack:
- Angular 21+ with @if/@for control flow
- Standalone Components with Signal-based state management
- Tailwind CSS v3 for styling
- HttpClient for data loading
- Default change detection (NOT OnPush - was causing scroll/animation issues)
- TypeScript with strict typing

     * zoomIn/Out()
     * fitToScreen()
     * resetPan()

   - Used by: docs.component.ts image modal
